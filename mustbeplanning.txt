# 🧱 PHASE 0 — Native Contract Layer (WAJIB, FONDASI)

> Tanpa ini, semua phase lain rawan mismatch

### 🎯 Tujuan

Mendefinisikan kontrak jelas antara Rust `.node` ↔ TypeScript

### 📦 Scope

* Definisikan semua method native:

  * `open(path)`
  * `close()`
  * `insert(collection, buffer)`
  * `find(collection, queryBuffer)`
  * `update`
  * `delete`
  * `aggregate`
  * `transaction_*`
* Tentukan format data:

  * ✅ Prefer: BSON / binary
  * ❌ Hindari JSON di hot path
* Definisikan async behavior:

  * Promise-based
  * Streaming (cursor)
* Definisikan error system:

  * code (string/enum)
  * message
  * metadata

### 📌 Output

* `native.d.ts` (type contract)
* `native.ts` (loader wrapper)

# ⚡ PHASE 5 — Performance Engineering

### 🎯 Tujuan

Native performance tanpa bottleneck TS

### 📦 Rules

* Hindari JSON stringify
* Async non-blocking
